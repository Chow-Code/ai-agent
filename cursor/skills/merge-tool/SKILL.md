# 文件分卷合并工具Skill

## 工具概述

文件分卷合并工具用于合并通过企业微信发送的分卷文件。工具支持`.part1`、`.part2`、`.part3`等格式的分卷文件，自动检测、分组和合并。

## 使用场景

- 收到企业微信发送的分卷文件需要合并时
- 需要合并多个分卷文件为一个完整文件时
- 需要验证分卷文件完整性时

## 前置条件

- 所有分卷文件已下载完成
- 分卷文件放在`tools/merge/files/`目录中
- 文件格式正确（`.part1`、`.part2`、`.part3`等）

## 命令格式

### 方式一：使用批处理文件（推荐）

```bash
cd tools/merge
./merge_parts.bat
```

### 方式二：直接运行可执行文件

```bash
cd tools/merge
./merge_files.exe
```

## 使用步骤

1. **准备分卷文件**：将收到的所有分卷文件（`.part1`、`.part2`、`.part3`...）放入`files`目录
2. **运行工具**：双击运行`merge_parts.bat`或运行`merge_files.exe`
3. **选择文件组**：如果有多个文件组，选择要合并的文件组
4. **确认合并**：工具会显示所有找到的分卷文件，确认无误后继续
5. **完成合并**：合并后的文件将保存在工具所在目录

## 输出说明

合并后的文件保存在工具所在目录，文件名根据分卷文件名自动确定。

例如：
- 分卷文件：`files/login.exe.pt00`、`files/login.exe.pt01`、`files/login.exe.pt02`
- 合并后文件：`merge/login.exe`

## 工具特性

1. **专用目录**：使用`files`子目录存放分卷文件，保持工作区整洁
2. **自动检测**：自动查找`files`目录下的所有分卷文件
3. **智能分组**：如果有多组分卷文件，会让用户选择要合并哪一组
4. **自动命名**：根据分卷文件名自动确定输出文件名
5. **排序合并**：按照正确的顺序合并所有分卷文件
6. **用户友好**：显示合并进度和结果信息

## 支持的文件格式

工具支持以下格式的分卷文件：

- `.part1`、`.part2`、`.part3`... 格式
- `.pt00`、`.pt01`、`.pt02`... 格式（企业微信常用格式）

## 常见问题

### Q: 找不到分卷文件？

A: 确保分卷文件放在`files`目录中，确保文件名格式正确（`.part1`、`.part2`等）。

### Q: 合并后文件损坏？

A: 确保所有分卷文件完整无损，确保按正确顺序合并（工具会自动排序）。

### Q: 合并过程中出错？

A: 检查磁盘空间是否足够，确保有写入权限。

### Q: 如何合并多组文件？

A: 工具会自动检测多组文件，会让用户选择要合并哪一组。可以多次运行工具合并不同的文件组。

## 最佳实践

1. **检查文件完整性**：合并前确保所有分卷文件都已下载完成
2. **使用专用目录**：将分卷文件放在`files`目录中，保持工作区整洁
3. **验证合并结果**：合并后验证文件是否完整和正确
4. **清理分卷文件**：合并成功后可以删除分卷文件
